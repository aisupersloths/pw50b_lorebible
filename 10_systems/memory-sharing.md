# Memory Sharing

## Canon

- Memories can be copied, branched, merged, archived, and restored.
- Passive duplication is near-lossless.
- Copying into active minds is more constrained than copying into passive media.
- Storage capacity and bandwidth remain important bottlenecks.
- Human-scale minds retain finite cognitive limits.
- Shared memories carry subjectivity and emotional residue.
- Full scrubbing is difficult.
- Memory-sharing has replaced some writing/video functions but has not necessarily replaced all of them.
- In high-tech societies, devices required for memory transactions are effectively ubiquitous.

## Implications

- Memory-sharing is infrastructure, not a novelty.
- Education, coordination, intimacy, labor, law, and entertainment are all shaped by it.
- Identity drift and archive management are permanent social issues.
