# Perchance Export Layer

This directory is for derived generator-facing exports.
It is not a source of canon.

## Purpose

Provide compact, atomic, Perchance-safe inputs such as:
- names
- factions
- social forms
- plot hooks
- world niche tags
- negative constraints

## Rules

- Export from canon; do not author canon here.
- Keep outputs small, typed, and atomic.
- Include negative constraints so generators avoid Earth leakage and other canon violations.
- Keep this layer easy for agents to ignore unless explicitly told to work on exports.
